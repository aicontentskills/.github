## 💡 Why These Skills?

AI video generators are powerful but fragmented:

- Individual clips are limited to 5–15 seconds, but real videos are 30s, 60s, 90s
- Every model has different optimal prompt structures
- Visual consistency across multiple generations is hard
- Post-production planning is usually an afterthought

These skills fill the **planning, prompting, and production** gaps so creators can ship professional-quality content without burning credits or wasting time.

- **[Windsurf](https://windsurf.com)** — use as `.windsurfrules`
- **ChatGPT / Claude.ai** — paste as custom instructions
- **Any LLM-based AI assistant** — the skill files are plain Markdown

## 🎯 Design Principles

1. **Tool-agnostic** — skills work with any modern AI video/content tool, not locked to a specific vendor
2. **Production-focused** — every skill output is actionable, copy-paste ready
3. **Opinionated about craft** — we favor cinematic and storytelling conventions over generic outputs
4. **Free and MIT licensed** — no lock-in, no trial, no catch
